this works for cmder

![](https://i.imgur.com/XZplVh5.png)

<br/>

## Installation

---

1. install the panda-theme
2. overwrite clink.lua

   `%cmder folder%/vendor/clink.lua`

3. overwrite npm.lua

   `%cmder folder%/vendor/clink-completions/npm.lua`

4. copy & paste my_prompt.lua

   `%cmder folder%/config`
